How to get an API key

  1. Log in to brieffill.com
  2. Go to Settings → API Keys
  3. Click "Generate" and give it a name (e.g. "Browser Extension")
  4. Copy the key (starts with bfk_) and paste it above
  5. Click "Test Connection" to verify
  6. Click "Save"

Usage

  • Select any text on a webpage → right-click → "Analyze with BriefFill"
  • Or click the extension icon and paste text manually
  • Use Ctrl+Shift+B to open the popup quickly
  • Your 20 most recent analyses are saved in the History tab
